ورقة المراجعة السريعة — English Exam

كل القواعد الأساسية في مكان واحد — الأزمنة · القواعد · الوظائف · نصائح القطعة

TENSES — جدول الأزمنة الكامل

النوع / Type Present — الحاضر Past — الماضي Future — المستقبل
Simple — البسيط I + V1 (s/es مع he/she/it) I + V2 I + will + V1
Continuous — المستمر I + am/is/are + V-ing I + was/were + V-ing I + will be + V-ing
Perfect — التام I + have/has + V3 I + had + V3 I + will have + V3
Perfect Cont. — التام المستمر I + have/has been + V-ing I + had been + V-ing I + will have been + V-ing
V1 = الفعل الأساسي (base)  |  V2 = الفعل الماضي (past)  |  V3 = التصريف الثالث (past participle)  |  V-ing = الفعل المضاف إليه -ing

كلمات دالة على الأزمنة

  • Present Simple: always, usually, every day, often
  • Past Simple: yesterday, ago, last year, in 2010
  • Present Continuous: now, at the moment, currently
  • Present Perfect: just, already, yet, ever, never, since, for
  • Future: tomorrow, next week, soon

قواعد سريعة للأزمنة

  • بعد will / modal → V1 مباشرة
  • بعد did → V1 مباشرة
  • بعد am / is / are → V-ing
  • بعد have / has → V3
  • بعد since / for → Present Perfect
  • He/She/It في Present Simple → V1 + s

القواعد العامة الأساسية

IF — الجملة الشرطية

Type 1 If + present → will + V1
مستقبل محتمل — If it rains, I will stay home.
Type 2 If + past → would + V1
خيالي/غير واقعي — If I were rich, I would travel.

Wish — أتمنى

القاعدة wish + past simple
لتمنّي عكس الواقع — I wish I had a car.
تنبيه wish + were (ليس was)
She wishes she were taller.

LINKING — أدوات الربط

because سبب I stayed home because it rained.
so نتيجة It rained, so I stayed home.
but تضاد I studied, but I failed.
although رغم Although it rained, I went out.

SO / SUCH

so so + adj/adv
It was so cold. / He ran so fast.
such such + (a/an) + adj + noun
It was such a cold day. / They are such kind people.
الفرق: so تليها صفة فقط — such تليها صفة + اسم

SINCE / FOR + Present Perfect

since بداية الوقت
I have lived here since 2010. (منذ عام ٢٠١٠)
for مدة زمنية
I have lived here for 5 years. (منذ ٥ سنوات)

PREPOSITIONS — حروف الجر

at وقت / مكان محدد
at 8 o'clock / at home / at night
on يوم / سطح
on Monday / on the table / on TV
in سنة / شهر / مكان
in 2024 / in January / in Iraq

TO + Infinitive

القاعدة to + V1 (الفعل الأساسي)
I want to study. / She likes to read.
  • بعد want / like / need / hope / plan → to + V1
  • بعد should / can / must / will → V1 بدون to

المبني للمجهول (Passive)

الصيغة be + V3
يُستخدم حين الفاعل مجهول أو غير مهم
The letter was written by Ali.
The bridge was built in 1990.
English is spoken worldwide.

FUNCTIONS — وظائف اللغة

🙋 Request — طلب مساعدة
Can you help me?
Could you help me, please?
Would you mind helping me?
هل يمكنك مساعدتي؟ / هل تمانع مساعدتي؟
🤝 Offer — عرض مساعدة
I can help you.
Shall I help you?
Would you like me to help?
أستطيع مساعدتك. / هل تريد مساعدتي؟
💡 Suggestion — اقتراح
Why don't we study together?
How about going to the library?
Let's review the grammar.
لماذا لا ندرس معاً؟ / ما رأيك بـ...؟
📌 Advice — نصيحة
You should revise every day.
You had better sleep early.
If I were you, I would ask the teacher.
ينبغي أن... / لو كنت مكانك...
🚪 Permission — طلب إذن
Can I go out, please?
May I use your pen?
Is it OK if I sit here?
هل يمكنني...؟ / هل يسمح لي...؟
✅ Agreement — موافقة
Yes, of course.
I agree with you.
That's right, exactly.
نعم بالطبع. / أوافقك الرأي. / صحيح تماماً.
❌ Refusal / Disagreement — رفض
Sorry, I can't.
I'm afraid I disagree.
I don't think so.
آسف، لا أستطيع. / أخشى أنني لا أوافق.
🙏 Apology — اعتذار
I'm sorry for being late.
I apologize for the mistake.
Never mind. / That's alright.
أنا آسف على... / لا بأس. / لا يهم.
⚠️ Warning — تحذير
Be careful! / Watch out!
Look out, there's a car!
You should be more careful.
كن حذراً! / انتبه! / توخَّ الحذر أكثر.
🎉 Congratulations — تهنئة
Congratulations on your success!
Well done! / That's great news.
I'm so happy for you.
تهانيّ! / أحسنت! / أنا سعيد جداً لأجلك.
😢 Sympathy — تعاطف
I'm sorry to hear that.
That's too bad.
I hope you feel better soon.
أتأسف لسماع ذلك. / هذا مؤسف. / أتمنى تحسّنك.
💬 Opinion — إبداء الرأي
I think / I believe ...
In my opinion, ...
As far as I know, ...
أعتقد أن... / في رأيي... / فيما أعلم...
نصيحة: ركّز على الكلمة الدالّة في السؤال (should = نصيحة / can = إمكانية / congratulations = تهنئة) تنبيه: احفظ الجمل كوحدات جاهزة، لا تترجم كلمة بكلمة

READING — نصائح القطعة والقواعد السريعة

📖 نصائح القطعة

  • اقرأ السؤال أولاً ثم القطعة لتعرف ما تبحث عنه
  • دوّر على نفس الكلمة في النص أو مرادفها — الجواب موجود في النص غالباً
  • لا تختار من فهمك العام فقط — استند إلى النص مباشرة
  • إذا سأل عن زمن → ابحث عن الكلمات الدالة (in, during, after, before)
  • إذا سأل عن مرجع ضمير → ارجع إلى الجملة السابقة
  • إذا سأل عن معنى كلمة → انظر السياق المحيط بها
  • إذا سأل عن الفكرة الرئيسية → اختر ما يمثّل كل الفقرة لا جزء منها

⚡ القواعد السريعة — Quick Rules

  • بعد will / modal → V1
  • بعد did → V1 (لا تستخدم V2)
  • بعد am / is / are → V-ing
  • بعد have / has → V3
  • بعد since / for → Present Perfect
  • so ≠ such — so + adj فقط / such + adj + noun
  • He/She/It → V1 + s في Present Simple
  • wish → past simple (وليس present)
  • المبني للمجهول: be + V3
  • الشرط Type1: if + present, will + V1
  • الشرط Type2: if + past, would + V1

🔤 أخطاء شائعة يجب تجنبها

  • لا تستخدم V2 بعد did → ❌ did went / ✅ did go
  • لا تنسَ -s مع he/she/it → ❌ She study / ✅ She studies
  • بعد should / can / will / must → V1 بدون to
  • لا تخلط بين so (نتيجة) و because (سبب)
  • لا تخلط بين since (بداية) و for (مدة)

📝 هيكل الإجابة المثالية

  • ابدأ بجملة مباشرة وواضحة
  • أضف سبباً أو مثالاً قصيراً
  • اختم بجملة تؤكد فكرتك
I prefer studying in the morning because I can focus better, so I finish my tasks faster.
