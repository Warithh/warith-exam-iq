TENSES — جدول الأزمنة الكامل

النوع / Type Present — الحاضر Past — الماضي Future — المستقبل
Simple — البسيط I + V1 (s/es مع he/she/it) I + V2 I + will + V1
Continuous — المستمر I + am/is/are + V-ing I + was/were + V-ing I + will be + V-ing
Perfect — التام I + have/has + V3 I + had + V3 I + will have + V3
Perfect Cont. — التام المستمر I + have/has been + V-ing I + had been + V-ing I + will have been + V-ing
V1 = الفعل الأساسي (base)  |  V2 = الفعل الماضي (past)  |  V3 = التصريف الثالث (past participle)  |  V-ing = الفعل المضاف إليه -ing

كلمات دالة على الأزمنة

  • Present Simple: always, usually, every day, often
  • Past Simple: yesterday, ago, last year, in 2010
  • Present Continuous: now, at the moment, currently
  • Present Perfect: just, already, yet, ever, never, since, for
  • Future: tomorrow, next week, soon

قواعد سريعة للأزمنة

  • بعد will / modal → V1 مباشرة
  • بعد did → V1 مباشرة
  • بعد am / is / are → V-ing
  • بعد have / has → V3
  • بعد since / for → Present Perfect
  • He/She/It في Present Simple → V1 + s

القواعد العامة الأساسية

IF — الجملة الشرطية

Type 1 If + present → will + V1
مستقبل محتمل — If it rains, I will stay home.
Type 2 If + past → would + V1
خيالي/غير واقعي — If I were rich, I would travel.

Wish — أتمنى

القاعدة wish + past simple
لتمنّي عكس الواقع — I wish I had a car.
تنبيه wish + were (ليس was)
She wishes she were taller.

LINKING — أدوات الربط

because سبب I stayed home because it rained.
so نتيجة It rained, so I stayed home.
but تضاد I studied, but I failed.
although رغم Although it rained, I went out.

SO / SUCH

so so + adj/adv
It was so cold. / He ran so fast.
such such + (a/an) + adj + noun
It was such a cold day. / They are such kind people.
الفرق: so تليها صفة فقط — such تليها صفة + اسم

SINCE / FOR + Present Perfect

since بداية الوقت
I have lived here since 2010. (منذ عام ٢٠١٠)
for مدة زمنية
I have lived here for 5 years. (منذ ٥ سنوات)

PREPOSITIONS — حروف الجر

at وقت / مكان محدد
at 8 o'clock / at home / at night
on يوم / سطح
on Monday / on the table / on TV
in سنة / شهر / مكان
in 2024 / in January / in Iraq

TO + Infinitive

القاعدة to + V1 (الفعل الأساسي)
I want to study. / She likes to read.
  • بعد want / like / need / hope / plan → to + V1
  • بعد should / can / must / will → V1 بدون to

المبني للمجهول (Passive)

الصيغة be + V3
يُستخدم حين الفاعل مجهول أو غير مهم
The letter was written by Ali.
The bridge was built in 1990.
English is spoken worldwide.

📌 Advice — نصيحة
You should revise every day.
You had better sleep early.
If I were you, I would study harder.
You ought to see a doctor.
Why not try a different method?
ينبغي أن... / من الأفضل لك... / لو كنت مكانك...
✔ الردّ: Thank you, that's good advice. / You're right, I'll try.
💡 Suggestion — اقتراح
Let's study together.
Why don't we go to the library?
How about / What about visiting the museum?
Shall we take a break?
What if we asked the teacher?
لنذهب معاً... / ما رأيك في...؟ / ألا نذهب...؟
✔ قبول: That's a great idea! / Sure, let's do it.
✔ رفض: I'd rather not. / I don't feel like it.
🙋 Request — طلب

Could you open the window?
Would you mind closing the door?
Please pass me the book.
I'd like you to check my work.
هل يمكنك...؟ / هل تمانع...؟ / أرجوك...
✔ Could أكثر أدباً من Can في الطلبات الرسمية
🤝 Offer — عرض مساعدة

✔ الردّ بالقبول: Yes, please. / That would be great.
✔ الردّ بالرفض: No, thank you. / I'm fine, thanks.
🚪 Permission — طلب إذن

✔ منح الإذن: Sure, go ahead. / Of course, no problem.
✔ الرفض: I'm afraid not. / Sorry, you can't.

✔ قبول: I'd love to! / That sounds great!
✔ رفض: I'm sorry, I'm busy. / Unfortunately, I can't make it.

✔ الردّ: You're welcome. / Don't mention it. / My pleasure. / Not at all.

💪 Ability — إبداء القدرة

You have to finish your homework.
It is necessary to drink water daily.
You need to see a doctor.

يجب أن... / ينبغي... / من الضروري... / لا بد من...
🚫 Prohibition — منع / نهي
You must not / mustn't smoke here.

It's not allowed to use phones in class.
No parking / No smoking.
It is forbidden to enter this area.
ممنوع أن... / لا يُسمح بـ... / محظور...
⭐ Preference — تفضيل
I prefer tea to coffee.
I'd rather stay home than go out.
I like reading more than watching TV.
I'd prefer to travel by train.

أفضّل... على... / أودّ بدلاً من... / أحبّ أكثر من...
⚠ I'd rather + V1 (بدون to): I'd rather go now.

This is unacceptable.

Could you please do something about this?

✔ الردّ: I'm sorry about that. / I'll look into it right away.

🎯 نصائح ذهبية لأسئلة الوظائف في الامتحان

  • ابحث عن الكلمة الدالة في الجملة — should = نصيحة | Let's = اقتراح | Can I = طلب أو إذن
  • إذا كان السؤال "What is the speaker doing?" → حدّد الوظيفة من السياق لا من الترجمة الحرفية
  • Would you like to = دعوة أو عرض — ركّز على السياق لتحديد أيهما
  • I'm sorry قد تعني اعتذاراً أو تعاطفاً — السياق يحدد المعنى
  • الردّ على الاقتراح بـ That's a great idea = قبول | بـ I'd rather not = رفض
  • احفظ جملة واحدة كاملة لكل وظيفة وستتعرف عليها في الامتحان بسهولة
